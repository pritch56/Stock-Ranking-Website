10. Live Trade Feed

Scrolling real-time feed.

Example:

BOT_X BUY BTC +0.4%
BOT_Y SELL AAPL -0.2%
BOT_Z BUY ETH +1.1%

Use WebSockets.

WebSocket Architecture

Backend sends updates.

FastAPI WebSocket
↓
Redis PubSub
↓
Frontend JS

Example JS:

const socket = new WebSocket("wss://site.com/ws");

socket.onmessage = (event) => {
   updateTradeFeed(JSON.parse(event.data));
};
